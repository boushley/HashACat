What is HashACat?

HashACat is the next generation of cloud computing! By combining cats with powerful cloud infrastructure we can provide a unique product, Cats as a Service (CaaS)

This web interface is boring!

Lucky you! There exists an even more boring way to get your hashed cat needs! HashACat provides a ReSTish interface to allow easy integration into every single product you can imagine!

Simply make a http request for hashacat.com/cat/<yourHash>, and a cat will follow! Or, for you JSON types, append ?﻿format=json to get a direct link to your selected cat

What if I want each unique hash and cat to be bound in perpetuity?

Bam, that's how it works. Each hash is mapped directly to one cat in our database of thousands. That hash will always grant the same cat, ensuring you never confuse your users by providing non-consistent cat imaging

Who owns all these cat pictures?

No idea. They are from all around the internet, and we at HashACat don't claim any sort of copyright over any of them.

Cross Domain biz

Yo dawg, we be supporting both JSONp and CORS for all your cross domain calling needs. CORS just works on the hash URL, and JSONp can be obtained by appending ?﻿format=json&callback=﻿<yourFunction> to the /cat URL
